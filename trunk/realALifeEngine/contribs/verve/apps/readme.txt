This directory contains several Verve applications.  Most can be built with SCons and Microsoft Visual Studio.  Some applications are extremely out of date and will not work; they are left here simply for historical reasons.  These include: "analyzer", "biped", "leaky neuron", and "reward prediction."


Learning Tests
--------------
A set of simple benchmark tasks.


Physical
--------
Physically simulated control tasks, including the pendulum swing-up task and the cart-pole/inverted pendulum task.  This application requires OPAL (http://opal.sourceforge.net), ODE (www.ode.org), and Ogre (www.ogre3d.org).
