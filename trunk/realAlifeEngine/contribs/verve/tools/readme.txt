This directory contains miscellaneous tools.  Each can be built with SCons.


Agent Image Creator
-------------------
Generates an image of an Agent's neural network connections from an Agent XML file.  Not working yet.


CCIPCA Test
-----------
Tests the Candid Covariance-Free Incremental Principal Components Analysis algorithm.


Doxygen
-------
The Doxygen script file to generate API documentation.


RBF Activation Sum
------------------
Used to help find the maximum sum of activation in a set of radial basis functions.


Value Image Creator
-------------------
Generates an image of an Agent's value function from a data file.
